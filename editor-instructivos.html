Bienvenido al Editor de Instructivos

Introducción

Este es un editor de texto enriquecido donde puedes crear y editar instructivos de manera fácil e intuitiva.

Características principales:

  • Editor WYSIWYG (What You See Is What You Get)
  • Formato de texto avanzado
  • Inserción de imágenes y enlaces
  • Guardado automático
  • Exportación a diferentes formatos

Cómo usar el editor

Simplemente comienza a escribir o utiliza la barra de herramientas para dar formato a tu texto.

Tip: Puedes usar las teclas de acceso rápido como Ctrl+B para negrita, Ctrl+I para cursiva, etc.
